Rohan's friend got invited for an event.But he got stuck in some other work so he told Rohan to attend the event on his behalf. Rohan agreed to attend the event  and went for the party with invitation pass. He goes to the Hotel but does not know the hall number. The manager tells him to enter the hall name on his pass in their computer and it will tell him the hall number.
Design a program that accepts Hall name as input and then checks whether the hall exists using contains() method. If the hall exist print the hall position(hall postition always starts from 0) using indexOf() method. And if the hall is not present print the message "Hall name does not exist".

Strictly adhere to the Object Oriented Specifications given in the problem statement. All class names, member variable names and function names should be the same as specified in the problem statement.

Create a class named Hall with the following private member variables
Data Type Variable Name
string name
double costPerDay
string owner

Include the appropriate constructor, getters, and setters for the Hall class.
Include the following member functions and constructors in the class Hall
Method Name Description
bool contains(list <Hall> halls, int n, string hallName) This method accepts list of halls, size and hall name to be searched as input. It checks whether the hallName is present in the list of halls, if yes then returns true else returns false
int indexOf(list <Hall> halls, int n, string hallName) This method returns the position of the hallName.
 
Create a driver class called Main. In the Main method, create a list of Hall objects and obtain information of halls and assign the values to the object's members using setters. Obtain the hall name to be searched by the user, call contains() method to check whether the obtained hall name is present in the list or not. If the hall name is present, call the indexOf() method to find the position of the hall name else print "Hall name does not exist".

Problem constraints:
Use the push_back() method to insert into the list and iterator to traverse the list.
Use begin() and end() to point first and the last element of the list

Input and Output Format :
Refer sample input and output for formatting specifications.
[All text in bold corresponds to input and the rest corresponds to output]
